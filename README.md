# رفيق العلاج

تطبيق React عربي RTL لتنظيم رحلة العلاج، المواعيد، الأدوية، الأعراض، التحاليل، وإرشادات الطبيب. يعمل محليًا ويحفظ البيانات في `localStorage`.

## التشغيل

```bash
npm install
npm run dev
```

## إنشاء نسخة إنتاجية

```bash
npm run build
npm run preview
```

> التطبيق منظم ومذكّر فقط ولا يقرر الجرعات أو المواعيد الطبية.

## الرابط العام

يُنشر الموقع تلقائيًا عبر GitHub Pages عند تحديث فرع `main`:

`https://zezomedo8080-max.github.io/rafiq-alilaj/`

البيانات التي يدخلها كل مستخدم تُحفظ محليًا في متصفحه ولا تُرسل إلى خادم.
