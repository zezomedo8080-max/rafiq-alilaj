# رفيق العلاج

تطبيق React عربي RTL لتنظيم رحلة العلاج، المواعيد، الأدوية، الأعراض، التحاليل، وإرشادات الطبيب. يحفظ نسخة محلية في `localStorage` ويدعم مزامنة مشفرة اختيارية بين الأجهزة.

## التشغيل

```bash
npm install
npm run dev
```

## إنشاء نسخة إنتاجية

```bash
npm run build
npm run preview
```

> التطبيق منظم ومذكّر فقط ولا يقرر الجرعات أو المواعيد الطبية.

## الرابط العام

`https://rafiq-alilaj.netlify.app/`

## المزامنة المشفرة بين الأجهزة

- اضغط أيقونة السحابة، ثم اختر **إنشاء وتفعيل المزامنة**.
- انسخ رمز المزامنة السري إلى الجهاز الآخر.
- على الجهاز الآخر، اختر **ربط جهاز موجود** والصق الرمز نفسه.
- تُشفّر البيانات داخل المتصفح باستخدام AES-GCM قبل إرسالها إلى Netlify Blobs.
- لا تشارك رمز المزامنة السري؛ فهو مفتاح الوصول إلى البيانات.

تظل هناك نسخة محلية على كل جهاز. عند تفعيل المزامنة، تُرسل نسخة مشفرة إلى التخزين السحابي. هذا التطبيق للاستخدام الشخصي والتنظيمي، وليس بديلًا عن أنظمة السجلات الطبية المعتمدة للمؤسسات.
