# رفيق العلاج

تطبيق React عربي RTL لتنظيم رحلة العلاج، المواعيد، الأدوية، الأعراض، التحاليل، وإرشادات الطبيب. يحفظ نسخة محلية في `localStorage` ويدعم مزامنة مشفرة اختيارية بين الأجهزة.

## التشغيل

```bash
npm install
npm run dev
```

## إنشاء نسخة إنتاجية

```bash
npm run build
npm run preview
```

> التطبيق منظم ومذكّر فقط ولا يقرر الجرعات أو المواعيد الطبية.

## الرابط العام

`https://rafiq-alilaj.netlify.app/`

## المزامنة المشفرة بين الأجهزة

- من الجهاز الذي يحتوي على البيانات الحالية، اضغط أيقونة السحابة ثم اختر **إنشاء الحساب ورفع البيانات**.
- اختر اسم حساب وكلمة مرور، ثم شارك الرابط واسم الحساب وكلمة المرور مع الشخص المصرح له فقط.
- على أي جهاز آخر، افتح الرابط ثم اختر **تسجيل الدخول لحساب موجود** واكتب نفس اسم الحساب وكلمة المرور.
- أي تعديل يتم حفظه في الحساب المشترك، وتستطيع الأجهزة الأخرى جلبه تلقائيًا أو عبر زر **مزامنة الآن**.
- تُشفّر البيانات داخل المتصفح باستخدام AES-GCM قبل إرسالها إلى Netlify Blobs.
- لا تشارك اسم الحساب وكلمة المرور إلا مع من تثق به، لأنهما مفتاح الوصول والتعديل على البيانات.
- إذا نسيت كلمة المرور لا يمكن استرجاع البيانات السحابية المشفرة.

تظل هناك نسخة محلية على كل جهاز. عند تفعيل المزامنة، تُرسل نسخة مشفرة إلى التخزين السحابي. هذا التطبيق للاستخدام الشخصي والتنظيمي، وليس بديلًا عن أنظمة السجلات الطبية المعتمدة للمؤسسات.
